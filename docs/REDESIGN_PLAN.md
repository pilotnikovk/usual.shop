## 4. UI/UX Рекомендации

### 4.1 Цветовая палитра (на основе референса METALLURG)
- **Основной (Primary):** #1E3A8A (синий) - надежность, профессионализм
- **Акцентный (Accent):** #F97316 (оранжевый) - CTA, активные элементы
- **Фон:** #F9FAFB (светло-серый)
- **Текст:** #1F2937 (темно-серый)
- **Белый:** #FFFFFF

### 4.2 Типографика
- **Заголовки:** Inter/Roboto Bold, 24-48px
- **Основной текст:** Inter/Roboto Regular, 14-16px
- **Подписи:** 12-14px, серый цвет

### 4.3 Компоненты UI
- Кнопки с закругленными углами (8px)
- Карточки товаров с тенью при наведении
- Иконки Font Awesome
- Модальные окна для форм
- Плавные анимации (transition 0.3s)

### 4.4 Адаптивность
- Desktop: 1280px+
- Tablet: 768px - 1279px
- Mobile: до 767px
- Гамбургер-меню на мобильных
- Адаптивные сетки (grid/flexbox)

### 4.5 CTA (Call-to-Action)
- Главный CTA: "Оставить заявку" (оранжевый)
- Вторичный CTA: "Перейти в каталог" (синий)
- Фиксированная кнопка звонка на мобильных
- WhatsApp виджет в углу
